# Number Guessing Game in Python
## 📍Description
This project is a beginner-level Python number guessing game. The computer randomly selects a number between 1 to 100, and the user must guess the correct number in the minimum number of attempts possible.

## 🚀Features
- Random number generated by the computer between 1 to 100
- User-friendly input to guess the number
- Hints provided after each guess (Too high / Too low)
- Counts the number of attempts taken by the user
- Simple logic using basic Python concepts (loops, conditions, variables)
- Beginner-friendly and easy to understand

### ⚙Technologies Used
-Python
-Random module
